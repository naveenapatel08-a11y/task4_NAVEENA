# task4_NAVEENA
# OCR Text Extraction using Python

## 📌 Project Overview

This project uses **Optical Character Recognition (OCR)** to extract text from images. It processes an image using **OpenCV** and **Pytesseract** and converts the detected text into machine-readable format.

## 🚀 Features

* Extracts text from image files
* Image preprocessing using OpenCV
* Converts images to grayscale
* Applies thresholding for better accuracy
* Displays extracted text in the terminal

## 🛠️ Technologies Used

* Python
* OpenCV (cv2)
* Pytesseract
* Tesseract OCR Engine

## ▶️ How to Run

1. Install required libraries:

```bash
pip install opencv-python
pip install pytesseract
```

2. Install Tesseract OCR on your system.

3. Run the program:

```bash
python PROJECT4.py
```
